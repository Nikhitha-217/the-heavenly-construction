## Hosting Instructions

### Option 1 — Hostinger / cPanel Hosting (Recommended)
1. Log in to your hosting control panel
2. Open **File Manager** → navigate to `public_html`
3. Upload ALL files and folders (keep folder structure intact)
4. Visit your domain — site is live!

### Option 2 — Netlify (Free Hosting)
1. Go to https://netlify.com → Sign up free
2. Drag and drop the entire `heavenly-website` folder
3. Your site is live in seconds with a free .netlify.app domain
4. Connect your custom domain in Settings → Domain Management

### Option 3 — Vercel (Free Hosting)
1. Go to https://vercel.com → Sign up
2. Click "Add New Project" → "Import" → upload folder
3. Deploy — done!

### Option 4 — GitHub Pages (Free)
1. Create a GitHub account and new repository
2. Upload all files to the repository
3. Go to Settings → Pages → Source: main branch
4. Site live at: https://yourusername.github.io/repository-name

## Custom Domain Setup
After hosting, point your domain:
- Add an A Record pointing to your host's IP
- Or add a CNAME record if using Netlify/Vercel
